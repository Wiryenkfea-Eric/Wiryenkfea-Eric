## About

I started in agriculture, transitioned into tech through grit and curiosity, and never looked back. I've deployed EKS clusters, automated infrastructure with Ansible & Terraform, and built CI/CD pipelines across AWS and Azure.

As an **AWS Community Builder**, I contribute to the global cloud community through content, mentorship, and events.

Today I engineer cloud-native systems by day and train the next wave of engineers as CIO of [Solavise Technologies Institute](https://solavisetech.com) (AWS Technical Partner).
